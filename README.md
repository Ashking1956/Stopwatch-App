# Stopwatch App
A simple stopwatch app built using Jetpack Compose in Android.

## Features
- Displays minutes, seconds, and milliseconds.
- Start, pause, and reset functionality.
- Clean and responsive UI.

## Technologies Used
- Jetpack Compose
- Kotlin
